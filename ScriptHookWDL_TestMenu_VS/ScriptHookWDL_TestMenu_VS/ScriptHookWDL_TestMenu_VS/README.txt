# ScriptHookWDL Test Menu (with VS Project)

Simple test mod menu for Watch Dogs Legion.

- Press F1 for mod menu popup.
- F2 for "Godmode" (stub), F3 for "Give Money" (stub).
- Log is written to ScriptHookWDL_log.txt in the same folder.

Build as x64 DLL, drop as ScriptHookWDL.dll in the game's bin folder.
