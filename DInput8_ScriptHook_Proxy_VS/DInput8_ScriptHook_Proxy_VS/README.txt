# dinput8.dll ScriptHook Proxy (with VS Project)

Drop-in dinput8.dll proxy loader for Watch Dogs Legion.
- Includes dinput8.sln (solution), dinput8.vcxproj (project), and main.cpp (source).
- Place your built dinput8.dll and ScriptHookWDL.dll in the game's bin folder.
- All log output goes to dllproxy_log.txt in the same folder.

Build as x64 DLL named dinput8.dll.
